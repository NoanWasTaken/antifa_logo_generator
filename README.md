# Antifascist Logo Generator

Customize an SVG antifascist logo in your browser. Add text on an arc, swap in your own image, adjust colors and layout — then download or copy the SVG.

[Try it](https://antifalogogenerator.noandelatouche.dev/)

## Usage

Open `index.html` in any browser, or run:

```sh
bun dev
```

## Stack

- Vanilla HTML, CSS, JS
- [opentype.js](https://github.com/opentypejs/opentype.js) for text-on-arc rendering
